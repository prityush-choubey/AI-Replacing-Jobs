
#  AI Replacing Jobs – Power BI Data Analysis Project

##  Project Overview
This project analyzes how Artificial Intelligence (AI) may replace or transform different job roles in the future.  
The analysis is based on skills, education level, AI exposure, and automation probability to identify:

- Jobs at high risk of automation  
- Jobs that are more resilient to AI  
- Skills that help roles stay future-proof  

The project is built using Power BI with a structured CSV dataset.

##  Project Objectives
- Analyze AI automation risk across job roles  
- Compare skill requirements for different professions  
- Understand the impact of education level on tech adaptability  
- Identify future-safe careers based on creativity, leadership, and analytical skills  


##  Dataset Information
- File Name: `ai.csv`
- Format: CSV
- Type: Curated / synthetic dataset for analysis

## Key Insights
- Jobs requiring creativity, leadership, and human interaction show lower automation risk  
- Highly repetitive roles have higher AI exposure  
- Higher education levels improve adaptability to AI-driven changes  

##  Business Use Case
- Students: Choose AI-resilient career paths  
- Job Seekers: Understand future automation risks  
- Organizations: Plan workforce upskilling strategies  
- Policy Makers: Design reskilling and employment programs
